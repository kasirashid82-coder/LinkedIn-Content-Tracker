## LinkedIn-Content-Tracker

## Overview

# This project allows structured tracking of LinkedIn content metrics to analyze performance trends over time.

## Built to:
	•	Log post-level performance data
	•	Compare engagement across posts
	•	Identify patterns in hooks, topics, and structure
	•	Support data-driven content decisions

## Features
	•	Manual input of LinkedIn ##metrics:
	•	Reach
	•	Engagement
	•	Link clicks
	•	Profile activity
	•	Structured post logging
	•	Organized metric display
	•	Basic performance comparison

## Tech Stack
	•	HTML
	•	CSS
	•	Vanilla JavaScript

##Purpose

# Instead of relying on intuition alone, this tracker enables structured analysis of:
	•	Hook strength
	•	Topic alignment
	•	Engagement patterns
	•	Content iteration performance

## Future Improvements
	•	Local storage persistence
	•	CSV export
	•	Filtering by topic/category
	•	Visual performance charts
